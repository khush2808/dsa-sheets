# DSA Sheets

Extracted problem data and static sheet browsers for:

- `strivers-a2z-sheet` - Striver's A2Z Sheet
- `striver-450-sheet` - alias to Striver's A2Z Sheet
- `neetcode-all` - NeetCode All
- `neetcode-250` - NeetCode 250
- `neetcode-150` - NeetCode 150
- `blind-75` - Blind 75

## Files

- `strivers-a2z-problems.json`
- `neetcode-problems.json`
- `sheets/strivers-a2z-problems.xlsx`
- `sheets/neetcode-problems.xlsx`

Serve the folder with any static server from the repository root and open `/strivers-a2z-sheet/`, `/neetcode-all/`, `/neetcode-250/`, `/neetcode-150/`, or `/blind-75/`.

# DSA Sheets

Extracted problem data and static sheet browsers for:

- `host/sheet-1` - Striver's A2Z Sheet
- `host/sheet-2` - NeetCode Sheet

## Files

- `strivers-a2z-problems.json`
- `neetcode-problems.json`
- `sheets/strivers-a2z-problems.xlsx`
- `sheets/neetcode-problems.xlsx`

Serve the folder with any static server from the repository root and open `/host/sheet-1/` or `/host/sheet-2/`.
